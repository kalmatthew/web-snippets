---
title: Spending Hit Dice
reference: Player's Handbook, page 186
---

- Hit dice determined by the class
- To spend one, roll it, add your Constitution modifier, add it to your health

| Class     | Hit Dice |
| --------- | -------- |
| Artificer | d8       |
| Barbarian | d12      |
| Bard      | d8       |
| Cleric    | d8       |
| Druid     | d8       |
| Fighter   | d10      |
| Monk      | d8       |
| Paladin   | d10      |
| Ranger    | d10      |
| Rogue     | d8       |
| Sorcerer  | d6       |
| Warlock   | d8       |
| Wizard    | d6       |
